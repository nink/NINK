# Nink

Grocery price intelligence — splash site and store flyer ingestion APIs for [nink.com](https://www.nink.com).

## What lives here

- `index.html` / `style.css` — public marketing site
- `api/ingest-walmart.js` — Walmart flyer ingestion
- `api/ingest-gianttiger.js` — Giant Tiger flyer ingestion
- `api/ingest-foodbasics.js` — Food Basics flyer ingestion
- `api/debug-foodbasics.js` — Food Basics debug endpoint
- `vercel.json` — deployment and cron schedule
